# Travel-Itinerary
This is a Japan Travel Itinerary Website for my Senior Project. 
This website will give users a user-friendly experience to help plan for future trips to Japan, whether it may be looking up best places to eat or see, or fun activities to do, and even recommend best places to stay or help you discover some hidden gems other tourists may overlook. 
The tools that I am using in this project are: Next.js, Next.js API Routes, Tailwind CSS, PostgreSQL, Prisma, Clerk, Supabase, Mapbox, ExchangeRate API, and Vercel.
